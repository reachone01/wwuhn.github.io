CPPPrimer和CPPPrimerPlus

一、出版时间不同

1、C++Primer Plus：是2015年人民邮电出版社e799bee5baa6e997aee7ad94e58685e5aeb931333431376637出版的图书，作者是 [美]普拉塔（Prata,S.）。

2、C++Primer：是2006年6月1日人民邮电出版社出版图书，作者Stanley B. LippmanBarbara E. Moo JoséeLaJoie 。

二、内容不同

1、C++Primer Plus：针对C++初学者，从C语言基础知识开始介绍，然后在此基础上详细阐述C++新增的特性，因此不要求读者有较多C语言方面的背景知识。本书可作为高等院校C++课程的教材，也可供初学者自学C++时使用。

2、C++Primer：内容是C++大师Stanley B. Lippman丰富的实践经验和C++标准委员会原负责人Josée Lajoie对C++标准深入理解的完美结合，已经帮助全球无数程序员学会了C++。

C++ Primer Plus适合入门，内容包含C语言和C++，有一些C++11特性

C++ Primer就全是C++，也包括C++11特性

c++ Primer是字典 （新版本会更侧重应用）

c++ Primer Plus是教程

C++Primer Plus是根e799bee5baa6e79fa5e98193e4b893e5b19e31333431363535据2003年的ISO/ANSI C++标准编写的，通过大量短小精悍的程序详细而全面地阐述了C++的基本概念和技术。

C++Primer新增大量教学辅助内容，用于强调重要的知识点，提醒常见的错误，推荐优秀的编程实践，给出使用提示，还包含大量来自实战的示例和习题。

《Plus》是一本烂书，在1991年的C++语言版本上插入了一些C++03内容，就成了2012新版书了。作者对很多C++的概念理解都是错的。

而Lippman是C++语言发展演进的核心人物，了解最新语言标准的来龙去脉与一切细节。无可撼动的权威之作。

更推荐plus，毕竟代码要写出来才能记住，理解才深刻，然而primer这点做的很差，给的都是代码片段，写个demo还得去网上找代码的上下文，plus这点做的好点，而且有不理解的代码贴出来了下面就告诉你为什么，primer适合当字典，plus适合边看边敲，至少我看plus收获比primer大的多的多。
